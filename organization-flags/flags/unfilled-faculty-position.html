Unfilled Faculty Position

1. Organization Unfilled Faculty Position Option

Operating fund or account currently has an unfilled or unassigned faculty position, where one is available. Only relevant for funds associated to faculty positions.

2. Found Unfilled Faculty Position Tag

In order for you to achieve compliance for fund, you must assign an approved faculty or staff to this fund as recipient.

Relevant Fields

  • Fund Type

3. Code

if (allowedComplianceStatus?.includes('UnfilledFacultyPosition')) {
	// Check if the fund type contains "Faculty" or "Academic Position" (case-insensitive)
	if (fund?.type && (/faculty/i.test(fund.type) || /academic position/i.test(fund.type))) {
		// Check if both faculty name and email are blank
		const isFacultyNameEmpty = !fund?.faculty?.firstName && !fund?.faculty?.lastName
		if (isFacultyNameEmpty) {
			statusList.push('UnfilledFacultyPosition')
		}
	}
}


if (allowedComplianceStatus?.includes('UnfilledFacultyPosition')) {
	// Check if the fund type contains "Faculty" (case-insensitive)
	if (fund?.type && /faculty/i.test(fund.type)) {
		// Check if faculty appointment start date is null
		if (fund?.faculty?.appointmentStartDate === null) {
			statusList.push('UnfilledFacultyPosition')
		}
		fund?.faculty?.appointmentEndDate
		// Additional condition: Check if the current date is greater than appointment end date
		const currentDate = new Date() // Get the current date
		// Check if appointmentEndDate is defined and not null
		if (fund?.faculty?.appointmentEndDate) {
			const appointmentEndDate = new Date(fund.faculty.appointmentEndDate)
			if (currentDate > appointmentEndDate) {
				statusList.push('UnfilledFacultyPosition')
			}
		}
	}
}
