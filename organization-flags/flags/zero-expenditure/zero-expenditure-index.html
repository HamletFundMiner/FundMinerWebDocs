Specific financial field that is being used to define or track whether an account or fund has incurred any expenses.
The 'expenses' selectable field is for choosing the variable that will represent the expenses flag.
If this variable flag stays 0 it will trigger in the fund the Zero Expenditure flag

1. Organization Zero Expenditure Option

Zero Expenditures: Total Capital Out of Compliance
'Track all funds that have $0 in total expenses year-to-date.'

"Zero Expenditure" refers to funds or accounts that have had no expenses recorded year-to-date. This means that throughout the current year, there have been no financial outflows or spending associated with these funds.

Tracking this option makes possible ensuring that the system identifies and monitors any accounts or funds that have not incurred any expenses so far this year. This could be useful for compliance or reporting purposes, where it is important to know which funds have remained untouched in terms of spending.

Optional Fields:

  • Reconfigure Zero Expense Variable

2. Found Zero Expenditure Tag

Operating fund or account has had no expenses during current fiscal year.


Usually by default you can find this trigger value for the Zero Expenditure tag from YTD Expenses by default or Reconfigure Zero Expense Variable if its already modified.
